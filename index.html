Abstract

Why 3D thinking matters for VLA

We propose a 3D-thinking-guided co-training framework that enables vision-language-action models to perform 3D spatial reasoning implicitly during action prediction. Our key observation is that 3D geometry perception and 3D spatial reasoning are distinct capabilities that should be injected at different feature hierarchies instead of being entangled in a single auxiliary loss.

During training, 3DThinkVLA couples three latent-space components: a geometry perception module for low-level spatial cues, an online 3D reasoning distillation module centered on a shared reasoning anchor token, and a spatially augmented action integration module that injects these priors into action-query tokens. At deployment, the method keeps only lightweight adapters and still operates purely on 2D images, avoiding external 3D sensors, explicit text reasoning, or architectural rewrites of the pretrained VLM.

This design delivers strong performance on LIBERO, LIBERO-PLUS, SimplerEnv, and challenging real-world manipulation while mitigating catastrophic forgetting and preserving the simplicity of standard image-based VLA inference.

Method

A three-stage latent co-training recipe

Perception

Latent 3D Geometry Perception

Align visual intermediate features with a 3D foundation model to import geometry priors into the pretrained VLM.

Reasoning

Online 3D Reasoning Distillation

Use a shared reasoning anchor token to transfer explicit teacher reasoning into implicit student-side spatial representations.

Action

Spatially Augmented Action Integration

Inject disentangled geometric and reasoning features into action-query tokens to improve policy grounding and suppress shortcut learning.

Core design principles

  • Separate geometry perception from 3D reasoning instead of treating them as the same supervision signal.
  • Keep the deployment path lightweight by discarding the teacher branch and 3D foundation model after training.
  • Transfer spatial cognition through latent tokens rather than text outputs, avoiding chain-of-thought overhead.
  • Preserve pretrained VLM knowledge by using adapter-style injection rather than invasive backbone changes.

Results

Where 3DThinkVLA gains come from

Generalization

Strong results on LIBERO and LIBERO-PLUS indicate that latent 3D priors improve task transfer beyond narrow single-scene memorization.

Embodied robustness

Performance on SimplerEnv and real-world manipulation suggests the learned priors remain useful when the policy faces harder geometry and viewpoint shifts.

Efficient deployment

The model runs with standard image inputs only, retaining the practical deployment footprint of conventional VLA systems while inheriting 3D-aware behavior.
