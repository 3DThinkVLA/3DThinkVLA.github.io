Abstract

We propose a 3D-thinking-guided co-training framework that enables vision-language-action (VLA) models to perform 3D spatial reasoning implicitly during action prediction. Our core insight is that 3D geometry perception and 3D spatial reasoning are distinct capabilities that can be disentangled and injected at different feature hierarchies.

During training, three tightly coupled components work in concert primarily within the latent space: a latent 3D geometry perception module aligns intermediate visual features with a 3D foundation model, an online 3D reasoning distillation module transfers high-level spatial thinking through a shared reasoning anchor token, and a spatially augmented action integration module injects these priors into action-query tokens.

At deployment, 3DThinkVLA retains only lightweight adapters and operates purely on 2D images without requiring 3D sensors, external foundation models, or explicit text generation, while achieving state-of-the-art results on LIBERO, LIBERO-PLUS, SimplerEnv, and challenging real-world manipulation tasks.

Method Overview

3DThinkVLA separates 3D capability learning into three complementary stages so that perception, reasoning, and action grounding can be trained without overwriting the pretrained VLM's semantic prior.

1. Latent 3D Geometry Perception

The model aligns intermediate visual features with a 3D foundation model to acquire low-level geometric cues while avoiding heavy architectural modifications.

2. Online 3D Reasoning Distillation

A shared reasoning anchor token carries explicit teacher-side 3D reasoning into the student VLA, allowing spatial thinking to be distilled without deployment-time chain-of-thought generation.

3. Spatially Augmented Action Integration

Geometric and reasoning priors are hierarchically injected into action-query tokens so the resulting policy remains spatially grounded and avoids shortcut behaviors.

Key Properties

The framework is designed to preserve pretrained knowledge, avoid catastrophic forgetting, and keep inference practical. The teacher branch and 3D foundation model are training-only components, so the final deployed system still accepts standard 2D observations.

Results

3DThinkVLA achieves state-of-the-art performance on LIBERO and LIBERO-PLUS, demonstrates strong embodied robustness on SimplerEnv, and transfers effectively to challenging real-world manipulation tasks.

These gains indicate that latent 3D priors improve both generalization and action grounding, while preserving the simplicity of image-only VLA deployment.
